备注 介绍
布局容器最外层的框框，HTML中就是最外层的DIV。(缩写 b)
卡片容器可见的div，标题中写文本也是可见的。(缩写 k)
卡片尾卡片容器的扩展，显示在卡片容器的底部位置。
自由卡片容器类似于卡片容器，内部由头中尾三个分组框构成，可自定义style(缩写 z)
宽窄容器横向两个DIV，左边的宽度占75%。
窄宽容器横向两个DIV，右边的宽度占75%。
表单容器1个标签+1个编辑框+1个按钮的组合美化框。
均分容器内部的所有组件都会被等宽分配。
按钮容器多个按钮的美化框。
卷帘容器内部可以防止任意组件，类似于QQ好友的分组一样切换效果。
短视频容器一个支持上下滑动视频的容器，群内下载案例。
视频容器一个播放视频的容器。
弹出层容器这个容器内部有从上到下3个子级分组框组合成，子级内部可以放任意组件。
分页容器内部由多个超级链接框构成。
展图容器内部由多个图片框构成。实现图片的展示，点击后可放大图片。需要在窗口中 【开启展图容器=真】
自由列表容器内部由多个分组框构成，分组框内可放置任意的组件。
提交容器再HTML表示from元素，不推荐使用。
