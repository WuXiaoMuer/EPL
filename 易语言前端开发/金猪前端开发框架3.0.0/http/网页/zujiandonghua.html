入场动画演示
支持的组件：分组框、按钮、图片框、标签。如果是设置分组框动画，那么分组框内的内的任意组件都会受到共同影响。
标记 值
动画触发鼠标、载入、入场
动画方式bounce flash pulse rubberBand shaka swing tada wobble jello等
动画循环真、假 默认假
动画入场延迟毫秒 默认50ms【动画触发=入场】有效
动画播放速度毫秒 默认500ms【动画触发=入场】有效
动画方式当【动画触发=入场】时，动画方式的名称会有变化。具体请看下方的入场方式。
入场方式 介绍

fade-down从上到下的滑入并且过程中有淡入效果

fade-left从右到左的滑入并且过程中有淡入效果
fade-up-right从左下角滑入并且过程中有淡入效果
fade-up-left从右下角滑入并且过程中有淡入效果
fade-down-right从左上角滑入并且过程中有淡入效果
fade-down-left从右上角滑入并且过程中有淡入效果
zoom-in由小变大淡入
zoom-in-up由小变大从下到上的滑入并且过程中有淡入效果
zoom-in-down由小变大从上到下的滑入并且过程中有淡入效果
zoom-in-right由小变大从左到右的滑入并且过程中有淡入效果
zoom-in-left由小变大从右到左的滑入并且过程中有淡入效果
zoom-out由大变小淡入
