标记说明
标记是【金猪易前端开发框架】的唯一美化页面的方式。掌握的标记越多，美化的效果就多好，在HTML开发中，标记就相当于是CSS。我们相当于把这些CSS进行了汉化。汉化的style样式通过标记的方式写入到HTML到起到美化页面的效果。【窗口标记】
背景图片=assets/img/300.jpg
背景颜色=rgb(220,220,220)
无缝背景=真或假 默认假
背景不动=真或假 默认假
网站介绍=这是网站的首页
网站关键词=百度,淘宝,京东
开启动画=假
开启滚图容器=假
开启展图容器=假
开启POST模块=假
开启主题模块=真
